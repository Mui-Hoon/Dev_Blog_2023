Week 07

1. DOM

  • As a browser loads a webpage, it creates a model of that page. The model is called a DOM Tree
  • Locate the node that represents the element you want to work with
    • getElementById()
    • querySelector()
    • getElementsByClassName()
    • getElementsByTagName()
    • querySelectorAll()
  • Use its text content, child elements, and attributes
    • className
    • classList
    • textContent
    • innerHTML

2. Event Handling

  • Events occur when users click or tap on a link, hover or swipe over an element, type on the keyboard, resize the window or when the page requested has loaded
  • When an event occurs, or fires, it can be used to trigger a particular function. Different code can be triggered when users interact with different parts of the page
  • Events can trigger the changes to the DOM. This is how a web page reacts to users

3. JSON Local Storage

  • HTML5 Storage is based on named key/value pairs.
  • Web storage (or HTML5 storage) lets you store data in the browser.
  • The data it stores can only be accessed by the domain that set the data
  • There are two types of storage: local and session storage.
  • implemented using the localStorage and sessionStorage objects.
  • localStorage.setItem(key, value)
  • localStorage.getItem(key)
