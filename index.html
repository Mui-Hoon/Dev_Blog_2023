Week 01

In this week, we learnt various html elements:

  • html - Defines the root of an HTML document
  • head - Contains metadata/information for the document
  • body - Defines the document's body
  • h1 to h6 - the various heading from biggest to smallest font size
  • p - paragraph
  • em - emphasis the text (italic)
  • strong - bold the text
  • ul - create an unordered list
  • ol - create an ordered list
  • li - define each list element
  • img - include an image
  • a - specify the hyperlink
  • span - define a section of document (inline)
  • div - define a section of document (block), usually used for a bigger section such as a new chapter
  • table - create a table (together with tr, th, td)
  • hr - horizontal rule, also defines a thematic change in the content
  • iframe - defines an inline frame

Week 02

In this week, we learnt

1. semantic html elements:

  • header - represents a container for introductory content or a set of navigational links. Typically contains: one or more heading elements, h1 to h6, logo or icon, authorship information
  • footer - Defines a footer for a document or section, Typically contains: authorship information, copyright information, contact information, sitemap, back to top links, related documents
  • nav - defines navigation links
  • article - defines an article
  • aside - for content that is indirectly related to the main content (sidebars)
  • section - defines a section in a document
  • details - Defines additional details that the user can view or hide
  • figure - Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc
  • form - Defines an HTML form for user input
  • mark - Defines marked/highlighted text
  • main - Specifies the main content of a document

2. css:

  • CSS - 3 ways to include styles in HTML
    • Inline style
    • Embedded style element in head section
    • External Stylesheet
  • Element selector
  • ID selector
  • Class selector
  • Pseudoclass selector – depending on the state of the link
  • External font

Week 03

In this week, we learnt

1. Form element – to create an HTML form for user input

  • Input element: for user input, display in many ways, depending on the type attribute (text, radio (use same name for same group), checkbox, email, password, date, time, file, submit, button, etc)
  • Label element: defines a label for the form elements
  • Select element: defines a drop-down list (use together with option elements that define the option within the drop-down list)
  • Textarea element: defines a multi-line input field (row and column attributes to specify the visible number of lines and width in a text area)
  • Button element: defines a clickable button

2. CSS Selectors

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)

3. Inheritance, specificity, cascading, !important

4. Box Model

content -> padding -> border -> margin

Normalize - a great reset sccript to include (use the link https://necolas.github.io/normalize.css/)

5. Media Queries

for small device: @mdeia (min-width: 576px) { css rule}

6. Positioning

  • position: relative, absolute, fixed
  • z-index (work ONLY on elements with a position property set to absolute, fixed, or relative)
  • An element with a higher z-index value overlaps an element with a lower z-index value.
  • Positioned elements have a z-index of 0 by default.

7. Flexbox

To start using the Flexbox model, you need to first define a flex container by setting the display property to flex

The flex container properties are:

  • flex-direction - defines the direction that the container stacks the flex items (row, row-reverse, column, column-reverse)
  • flex-wrap - specifies whether the flex items should wrap or not (wrap, nowrap – default, wrap-reverse)
  • flex-flow - shorthand for setting both flex-direction and flex-wrap
  • justify-content - used to align the flex items (center, flex-start – default, flex-end, space-around, space-between)
  • align-items - used to align the flex items (center, flex-start, flex-end, stretch, baseline)
  • align-content - used to align the flex lines.
