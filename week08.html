Data Attributes

  • We can use the data attribute to allow custom information to be added on to our HTML
  • Created using data-<custom name>
  • By using data attributes, we can use it to define variables for ourselves. Based on those “variable” settings, we can use it to further modify our elements to provide additional interactivity

API

  • AJAX (Asynchronous JavaScript And XML) is a technique for loading data into part of a page without having to refresh the entire page. The data is often sent in a format called JavaScript Object Notation (or JSON)
  • How AJAX Works
    • Create an XMLHttpRequest
      let XHR = new XMLHttpRequest()
    • Create a callback function
      XHR.onreadystatechange = function(){ 
    if (XHR.readyState == 4 && XHR.status == 200){ 
        console.log(XHR.responseText); 
    } 
};
    • Open a request
      XHR.open("GET", "https://api.github.com/zen");
    • Send the request
      XHR.send();
  • Using fetch()
      fetch(url) 
  .then(function(res){ console.log(res); })
  .catch(function(error){ console.log(error); })
